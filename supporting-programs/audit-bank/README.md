# Audit Bank

To keep the ecosystem secure, SDF covers essential audits for SCF-awarded Projects building Stellar's smart contracts. Learn more about the program [here](official-rules.md).&#x20;

**To get started:**

1. Check out if [you're eligible](official-rules.md#participant-eligibility)!
2. Fill out the application form for the Soroban Security Audit Bank received by email from the Stellar Community Fund.

If for some reason you're not able to find the information, reach out to communityfund@stellar.org or a member from the SDF team.&#x20;
